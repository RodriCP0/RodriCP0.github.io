§ 01 — manifesto

A web ficou cheia de sites iguais. Templates, frameworks, atalhos. Este não. Aqui, cada animação é matemática escrita à mão, cada som é uma onda sintetizada no momento, e nada — nada — foi importado de uma biblioteca. HTML. CSS. JavaScript. Só.
