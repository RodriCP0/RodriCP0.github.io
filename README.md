# RodriCP0.github.io

- **/** — Site Slovenian Mobility Network
- **/social/** — [Nexo](https://rodricp0.github.io/social/), uma rede social demo 100% client-side (feed, perfis, seguidores, mensagens, notificações, tema claro/escuro), com persistência em `localStorage`.
- **/o-site/** — [O SITE](https://rodricp0.github.io/o-site/), uma experiência imersiva sem dependências: aurora generativa em canvas, partículas interativas, arte de flow field (exportável em PNG), motor de física, sequenciador Web Audio e tipografia variável viva. Tema claro/escuro, cursor custom e easter egg Konami.
